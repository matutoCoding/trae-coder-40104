## 1. 产品概述

台球室计费叫台系统——面向台球厅经营者的智能排台与阶梯计费 Web 应用。解决传统手写排台易冲突、计费不透明、散场拆分混乱等痛点，实现球台资源可视化管理、按消费时长逐档累加计费、连续占用自动合并与中途散场灵活拆分。

目标用户：台球厅前台收银员、店长；核心价值：减少排台冲突、杜绝计费争议、提升翻台效率。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 登录方式 | 核心权限 |
|------|----------|----------|
| 店长 | 账号密码 | 全部功能 + 阶梯档位配置 + 球台建档 + 数据统计 |
| 前台 | 账号密码 | 开台/叫台/结账/占用拆分 + 账单查看 |

### 2.2 功能模块

1. **大厅总览页**：球台状态总览、快捷开台、实时计时、档位临界提示
2. **阶梯计费页**：档位维护、用量逐档计价模拟、计费规则展示
3. **账单管理页**：账单生成、历史账单、明细查看、退款标记
4. **球台排期页**：球台资源时间线、连续时段合并占用、占用区间拆分、包厢与散台区分

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 大厅总览 | 球台状态卡片 | 显示每张球台实时状态（空闲/占用/预约），已用时长，当前档位，预计下一档位时间 |
| 大厅总览 | 快捷开台面板 | 选择球台、输入客户信息、选择包厢/散台类型、确认开台并开始计时 |
| 大厅总览 | 档位临界提示 | 当消费时长接近下一档位时（提前5分钟），球台卡片闪烁警示并显示即将跳档信息 |
| 阶梯计费 | 档位列表 | 展示所有阶梯档位（起止时长、单价），支持增删改排序 |
| 阶梯计费 | 计费模拟器 | 输入消费时长，实时展示各档位用量与费用明细，逐档累加计算总价 |
| 阶梯计费 | 档位规则说明 | 以可视化图表展示阶梯计费规则，帮助前台向顾客解释 |
| 账单管理 | 当前账单 | 正在消费中的球台实时账单，展示已产生费用与档位明细 |
| 账单管理 | 结账生成 | 结账时自动生成完整账单，含各档位用量、单价、小计、总计 |
| 账单管理 | 历史账单 | 按日期/球台/客户筛选历史账单，支持查看详情 |
| 球台排期 | 时间线视图 | 以时间线方式展示每张球台当天占用情况，支持拖拽预约 |
| 球台排期 | 合并占用 | 同一局连订相邻时段自动合并为整段占用，视觉上为一整块 |
| 球台排期 | 拆分释放 | 中途散场时拆分占用区间，释放后半段供其他客户预约 |
| 球台排期 | 球台建档 | 新增/编辑球台信息（编号、类型：包厢/散台、状态、费率组） |

## 3. 核心流程

### 3.1 开台-计费-结账流程

1. 前台在总览页点击空闲球台「开台」
2. 输入客户信息，选择台类型（包厢/散台），确认开台
3. 系统开始计时，按第一档单价计费
4. 消费时长跨越档位临界时，系统弹出临界提示
5. 客户散场时，前台点击「结账」
6. 系统按阶梯规则逐档计算费用，生成账单
7. 确认收款后账单归档

```mermaid

    "选择空闲球台" --> "输入客户信息"
    "输入客户信息" --> "确认开台"
    "确认开台" --> "开始计时(第一档)"
    "开始计时(第一档)" --> "跨越档位临界"
    "跨越档位临界" --> "弹出临界提示"
    "弹出临界提示" --> "继续消费"
    "继续消费" --> "跨越档位临界"
    "继续消费" --> "客户散场"
    "客户散场" --> "逐档计费生成账单"
    "逐档计费生成账单" --> "确认收款"
    "确认收款" --> "释放球台"
```

### 3.2 占用合并与拆分流程

1. 客户A预约10:00-12:00使用1号台
2. 客户A续时12:00-13:00
3. 系统自动合并为10:00-13:00整段占用
4. 客户A在11:30散场
5. 前台执行「散场拆分」，系统将10:00-13:00拆为10:00-11:30（已占用）+ 11:30-13:00（释放）
6. 11:30-13:00可被其他客户预约

```mermaid

    "续订相邻时段B" --> "自动合并为整段"
    "自动合并为整段" --> "中途散场"
    "中途散场" --> "拆分占用区间"
    "拆分占用区间" --> "前半段标记已消费"
    "前半段标记已消费" --> "后半段释放可预约"
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深墨绿 (#0A1F1C) + 金色点缀 (#D4A843)，营造台球厅高端质感
- **辅色调**：深色木纹棕 (#3E2723)、丝绒红 (#8B1A1A) 用于包厢标识
- **按钮风格**：圆角胶囊按钮，主操作金色填充，次操作墨绿描边
- **字体**：标题用 Playfair Display 衬线体，正文用 Noto Sans SC
- **布局风格**：左侧导航 + 右侧内容区，球台卡片网格布局
- **图标**：线性图标 + 圆形背景，颜色与球台状态联动（绿=空闲、红=占用、黄=预约）

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 大厅总览 | 球台状态卡片 | 网格布局，卡片含球台编号、类型标签、计时器、档位进度条、快捷操作按钮 |
| 大厅总览 | 快捷开台面板 | 右侧抽屉式弹出，表单含客户名、手机号、台类型选择、费率组选择 |
| 大厅总览 | 档位临界提示 | 卡片边框金色脉冲动画 + 顶部浮动提示条 |
| 阶梯计费 | 档位列表 | 表格形式，每行含序号、时长区间、单价、编辑/删除按钮，底部新增按钮 |
| 阶梯计费 | 计费模拟器 | 左侧时长输入滑块 + 右侧实时费用柱状图（各档位分色） |
| 阶梯计费 | 规则说明 | 水平阶梯条形图，每段标注时长范围和单价 |
| 账单管理 | 当前账单 | 紧凑卡片列表，每卡含球台号、客户、已消费时长、当前费用 |
| 账单管理 | 结账生成 | 模态框，展示完整账单明细表格 + 总计金额 + 收款确认按钮 |
| 账单管理 | 历史账单 | 筛选栏 + 表格列表，支持日期范围、球台、客户筛选 |
| 球台排期 | 时间线视图 | 纵轴球台列表 + 横轴时间刻度，占用块按客户着色，拖拽创建预约 |
| 球台排期 | 合并/拆分 | 右键菜单或点击占用块弹出操作面板，含「续时合并」「散场拆分」选项 |
| 球台排期 | 球台建档 | 模态框表单，含编号、类型(包厢/散台)选择、关联费率组、状态开关 |

### 4.3 响应式设计

桌面优先设计，适配1280px及以上宽度。平板端（768-1279px）球台卡片缩为两列，时间线视图压缩。移动端（<768px）仅保留球台状态列表和快捷操作。
